🚀 Ejercicios Prácticos: Programación Orientada a Objetos (POO)
Este ejercicio contiene dos ejemplos fundamentales de cómo utilizar clases en Python para modelar situaciones del mundo real. Ambos ejercicios demuestran el uso de atributos, métodos y lógica de validación.

🏦 1. Sistema de Cuenta Bancaria
Este módulo define la clase CuentaBancaria, diseñada para gestionar las finanzas de un usuario de manera segura y organizada.

Características:
Gestión de Saldo, Validación de Fondos, Encapsulamiento Lógico.

Métodos Principales:
depositar(cantidad)
retirar(cantidad)
consultarSaldo()
mostrarInformacion()

🐾 2. Simulador de Mascotas Virtuales
El módulo Mascota modela el comportamiento y el bienestar de un animal doméstico mediante un sistema de puntos de felicidad.

Características:
Sistema de Felicidad, Límite de Estado, Evaluación de Bienestar.

Métodos Principales:
alimentar().
jugar()
esFeliz()
mostrarEstado()

🛠️ Tecnologías Utilizadas
Lenguaje: Python 3.x
