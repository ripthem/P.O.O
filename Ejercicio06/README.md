🛠️ Sistema de Herramientas (POO en Python)

Este proyecto implementa un sistema de herramientas inspirado en videojuegos (tipo Minecraft), utilizando Programación Orientada a Objetos (POO) en Python.

Se hace uso de:

Clases abstractas
Herencia
Encapsulamiento
Polimorfismo

📌 Descripción

El sistema define una clase base abstracta llamada Herramienta, de la cual heredan distintas herramientas como:

🗡️ Espada
⛏️ Pico
🪣 Pala
🏹 Arco

Cada herramienta tiene:

Un material
Una durabilidad
Un comportamiento específico al usarse

📌 Notas
Si la herramienta está rota, no puede usarse.
Cada uso reduce la durabilidad.
El daño depende del material.
El arco además depende de la cantidad de flechas.
