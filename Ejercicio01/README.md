# 📘 Ejercicio 01 - Clase Estudiante (POO)

## 📖 Descripción

Este ejercicio implementa una clase llamada `Estudiante` utilizando el paradigma de **Programación Orientada a Objetos (POO)** en Python.

El objetivo es modelar a un estudiante mediante atributos y métodos que permitan gestionar su información personal y sus calificaciones.


## 🎯 Objetivos del Ejercicio

* Aplicar el uso de clases y objetos.
* Implementar un constructor (`__init__`).
* Manipular atributos de instancia.
* Crear métodos para procesar información (como el cálculo de promedios).


## 🛠️ Tecnologías Utilizadas

* Lenguaje: **Python**


## 📌 Conceptos Aplicados

* Clases y objetos
* Encapsulamiento básico
* Listas en Python
* Métodos de instancia
* Cálculo de promedios
